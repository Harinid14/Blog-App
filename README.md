Simple Blog App with Local Storage.

Objective:
Develop a blog application that allows users to create, read, and delete posts, with the data stored locally using local storage.

Steps: Create a user interface for the blog app with options to create, view, and delete posts.
1. Use local storage (or an array) to store and retrieve blog posts on the client side.
2. Implement functionality to add new posts and save them in local storage.
3. Display the list of saved posts for users to read.
4. Add functionality to delete selected posts and update the local storage accordingly.
5. Test the application to ensure posts are properly saved, displayed, and deleted as expected.
